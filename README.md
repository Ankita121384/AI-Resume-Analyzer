# 📄 AI Resume Analyzer

An AI-powered resume analyzer built with Python, Streamlit, and Groq AI that provides instant feedback to help you land your dream job!

## 🌐 Live Demo
👉 [Click here to try the app](https://ai-resume-analyzer-ebfmy82xxlbctrghmaerex.streamlit.app)

## ✨ Features
- 📤 Upload resume in PDF or DOCX format
- 📊 Get an AI-generated resume score out of 100
- ✅ Identify strengths in your resume
- ⚠️ Get areas for improvement
- 🛠️ Discover missing skills for your target role
- 💡 Receive actionable suggestions
- 🎯 Get interview preparation tips

## 🛠️ Tech Stack
- **Python** — Core programming language
- **Streamlit** — Web interface
- **Groq AI (LLaMA 3)** — AI analysis engine
- **PyMuPDF** — PDF text extraction
- **python-docx** — DOCX text extraction
- **python-dotenv** — Environment variable management

## 🚀 How to Run Locally

1. Clone the repository:
git clone
https://github.com/Ankita121384/AI-Resume-Analyzer.git
2. Install dependencies:
pip install -r requirements.txt
3. Create a `.env` file and add your Groq API key:
GROQ_API_KEY=your_key_here
4. Run the app:
streamlit run app.py
